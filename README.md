# Phishing Email Analysis

## Project Goal
This project demonstrates a safe, legal, home-lab style phishing email analysis workflow for cybersecurity portfolio purposes.

## Scope
The project focuses on:
- identifying phishing indicators
- analyzing suspicious email structure and content
- documenting findings in a professional format
- building a GitHub-ready cybersecurity portfolio project

## Safety Rules
This repository is built only in a safe and legal environment.
It does NOT involve:
- sending phishing emails
- interacting with malicious infrastructure
- using stolen or private data
- targeting real people or organizations

## Project Structure
- `docs/` - methodology and notes
- `reports/` - finished analysis reports
- `samples/raw/` - original sample emails (only if safe and legal)
- `samples/sanitized/` - redacted/anonymized versions of samples
- `tools/` - helper scripts or utilities if needed later
